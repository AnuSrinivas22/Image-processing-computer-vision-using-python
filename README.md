# Digital Image Processing Lab

This repository contains Digital Image Processing lab programs implemented using Python, OpenCV, and NumPy. The programs demonstrate fundamental image processing techniques such as image enhancement, filtering, segmentation, edge detection, and morphological operations.

## Programs Included

1. Image Reading and Display
2. Image Histogram and Histogram Equalization
3. Image Thresholding
4. Image Filtering (Smoothing and Sharpening)
5. Edge Detection (Sobel, Canny)
6. Morphological Operations (Erosion, Dilation)
7. Image Segmentation
8. Image Transformation
9. Noise Removal Filters
10. Color Space Conversion (RGB to Grayscale, HSV)

## Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib

## Folder Structure
- programs → Contains all lab programs
- input_images → Input images used for processing
- output_images → Result images after processing

## Applications
Image processing techniques are used in:
- Medical image analysis
- Face detection
- Object detection
- Autonomous vehicles
- Satellite image processing
- Security and surveillance
